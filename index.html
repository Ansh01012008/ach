📋 One-Time Setup Steps
  1. Go to emailjs.com → Sign up free
  2. Click Email Services → Add Service → Choose Gmail / Outlook / Yahoo → Connect your studio email → Copy the Service ID
  3. Click Email Templates → Create Template → Use the template below → Copy the Template ID
  4. Click your name (top right) → Account → Copy the Public Key
  5. Paste all three below → Click Save Settings
